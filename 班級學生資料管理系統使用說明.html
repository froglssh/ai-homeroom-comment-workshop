班級學生資料管理系統使用說明

本系統用來集中管理一個班級中每位學生的基本資料、學生自評、導師評語、幹部與小老師、特殊表現、成績、學測模擬考、社團活動與專題研究資料。它適合導師在學期中持續累積資料，期末再快速整理成可列印、可匯出 Excel、可備份還原的學生資料表。

主程式線上網址：
https://froglssh.github.io/class-student-data-management-online/

資料存在本機瀏覽器 支援 Excel / CSV / DOCX / JSON 匯入 可分學期管理 可列印與匯出 Excel 可用 AI 潤飾評語
重要提醒：這個程式是瀏覽器端單頁工具，資料主要儲存在目前瀏覽器的 localStorage。換電腦、換瀏覽器、清除瀏覽器資料或使用無痕模式，都可能看不到原本資料。請養成定期按「備份班級」匯出 JSON 的習慣。

一、建議使用流程

  1. 先開啟主程式：進入主程式線上網址，建議使用電腦版 Chrome、Edge 或 Safari。
  2. 填寫班級名稱與導師：在左側上方輸入班級名稱與導師姓名，之後列印與匯出會自動帶入。
  3. 建立學生名冊：按「匯入資料」匯入學生名冊 Excel，或按「新增學生」手動新增。
  4. 選取學生：左側名冊點選某位學生，右側會顯示該學生的完整資料表。
  5. 依分頁填資料：依序填寫基本資料、職位工作、導師評語、學生自評、特殊表現、成績、模考、社團與專題研究。
  6. 使用匯入功能減少手打：可匯入通訊錄、Google 問卷、自評表、導師評語、段考成績、模考成績、社團資料與 JSON 備份。
  7. 整理評語草稿：在「評語草稿暫存區」按「整合本學期資訊」，再視需要用 AI 潤飾或複製到 AI 網站修改。
  8. 列印或匯出：用「列印」、「匯出 Excel」、「匯出全班」輸出正式資料。
  9. 定期備份：每次大量修改後，按「備份班級」下載 JSON，避免資料遺失。

二、畫面區域說明

左側欄：班級與名冊

左側欄是班級層級的操作區，包含班級名稱、導師、搜尋學生、學生名冊，以及匯入、備份、新增、清空等按鈕。平常切換學生時，主要都在這裡操作。

右側主區：學生資料表

右側顯示目前選取學生的資料。上方工具列負責 AI 設定、匯入、列印與匯出；下方則是該學生的各資料分頁與評語草稿區。

畫面元素用途操作方式
班級名稱記錄目前班級，例如高二仁、高三數理資優班。直接在左側輸入框輸入，系統會自動儲存。
導師記錄導師姓名，列印與匯出時會出現在表頭。直接在左側輸入框輸入，系統會自動儲存。
搜尋姓名 / 座號在學生名冊中快速找到學生。輸入姓名或座號的一部分，名冊會即時篩選。
學生名冊顯示目前班級所有學生，包含座號、姓名與資料完整度百分比。點學生列即可切換到該學生資料表。
資料完整度百分比用來粗略判斷該學生資料是否已填寫足夠。隨資料填寫自動更新；它不是正式成績，只是填寫進度提示。
學生姓名與座號右側學生頁面上方可直接修改姓名與座號。點文字後直接編輯，離開欄位時自動儲存。
性別選單記錄男、女或空白。在學生頁面上方選擇，供資料整理與輸出參考。
檢視學期切換「全部」或高一上到高三下。選「全部」可看完整資料；選某學期時，列印與匯出會依目前學期篩選。

四、每一個資料分頁的功能

一、基本資料

用來記錄學生個人與家庭聯絡資訊。預設欄位包含手機、住址、身分證字號、生日、畢業學校、備註、父姓名、父手機、母姓名、母手機。匯入通訊錄或基本資料 Excel 後，這些欄位會自動填入。

操作說明
直接輸入在每個欄位打字，系統會自動儲存。
＋ 欄位新增基本資料欄位，例如住家電話、緊急聯絡人、特殊身分註記。
刪除欄位點欄位旁刪除符號，會刪除所有學生此欄內容。

二、職位工作

記錄學生在各學期擔任的幹部、小老師或其他職務。預設欄位是「幹部」與「小老師」，列是高一上到高三下六個學期。

功能說明
幹部下拉選單可快速選班長、副班長、風紀、學藝、衛生、總務、康樂、圖資、輔導、保健等職務。
小老師下拉選單可快速選國文、英文、數學、物理、化學、生物、地科、地理、歷史、公民、音樂、美術、資訊、生科、體育等小老師。
手動輸入若職務不在選單內，可直接在儲存格輸入。
＋ 欄可新增「服務隊」、「自治幹部」、「校級職務」等欄位。

三、導師評語

用來整理各學期的正式評語資料。預設欄位包含評語代號、文字描述評語、日常表現、特殊表現。這一頁是期末產生成績單評語時最重要的分頁。

功能說明
評語代號可填入學校德性評語代號或代碼文字。
文字描述評語放正式敘述式評語，可由評語草稿暫存區填入。
日常表現記錄生活常規、出缺勤、態度、人際互動等摘要。
特殊表現記錄適合放入導師評語中的特殊成果或優良表現。
評語詞庫可選擇目標學期與目標欄位，再點選正向或待加強詞語，系統會自動插入該欄。

四、學生自評

用來收集學生自己填寫的文字材料。核心欄位包含個性、性情、向學、進取、紀律、交友、服務、擅長；補充欄位包含學校服務、社團表現、學業表現、生活常規、校內其他特殊表現、社會服務、活動表演、科學展覽、校外其他特殊表現與備註。

功能說明
各學期自評每個學期都有獨立資料，不會互相覆蓋。
核心描述適合放學生對自己特質與學習態度的敘述。
自評補充適合放 Google 問卷、學生自評表或活動記錄中較具體的事實。
＋ 補充欄位可加入原問卷中的新題目，例如「自主學習成果」、「志工服務」。

五、特殊表現

記錄各學期的特殊表現與獎懲記錄。這裡適合放正式可查證的校內外表現，例如競賽、服務、表演、獎勵、懲處、班級活動貢獻等。

若之後要讓評語草稿自動整合特殊表現，建議用具體事件寫法，例如「參加校內科展獲佳作」、「協助班級園遊會器材管理」，不要只寫「表現很好」。

六、成績表現

記錄一段考、二段考、期末考的班排與類組排，也會整理較強科目與較弱科目。若使用段考成績 Excel 匯入，系統會自動判斷 80 分以上為較強科目，含 # 或不及格標記者為較弱科目，並整合到此分頁。

欄位說明
一段考班排 / 類組排記錄第一次段考排名。
二段考班排 / 類組排記錄第二次段考排名。
期末考班排 / 類組排記錄期末考排名。
較強科目可手動填寫，也可由段考匯入自動整理。
較弱科目可用於提醒需要加強的科目；輸出評語時會作為參考。

七、學測模考

記錄第一次、第二次、第三次學測模擬考。欄位包含國文、英文、數學A、數學B、社會、自然、總級分、總分、班排名、校排名、全體排名與備註。這個分頁不受「檢視學期」篩選，因為模考是以考次為主。

八、社團活動

記錄學生每學期參與的社團、職稱與社團表現。可透過社團資料 Excel 批次匯入，也可手動補寫。若學生是社長、幹部、活動負責人或有成果發表，建議寫在「社團表現與活動」。

九、專題研究（科展競賽）

記錄學生參與專題研究、科展、競賽、活動與獎懲記錄。若學生有研究題目、擔任角色、得獎紀錄或跨校活動成果，可集中放在這一頁，方便日後寫推薦、學習歷程回饋或導師評語。

十、自訂項目

按「新增其他項目」後，可在九大分頁後新增自訂分頁。建立時要輸入項目名稱與欄位名稱，例如「校外營隊」包含活動名稱、擔任角色、成果；或「證照檢定」包含證照名稱、通過日期、備註。自訂項目同樣支援列印與 Excel 匯出。

五、匯入資料功能完整說明

按「匯入資料」或「匯入」後，會出現匯入視窗。系統會依你選擇的資料類型讀取 Excel、CSV、DOCX 或 JSON，並讓你確認欄位對應後再寫入。

匯入類型適合資料匯入後寫入位置操作重點
學生名冊 Excel含多個班級分頁的學生名冊。建立每位學生分頁，寫入座號與姓名。選班級分頁，確認座號欄與姓名欄，可勾選匯入前清空現有學生。
全班基本資料 Excel/CSV通訊錄、校務系統匯出的基本資料。基本資料分頁。系統會找表頭並猜測姓名、座號、性別、住址、電話、父母資料；可手動調整欄位對應。
之前導師評語 Excel舊評語、德性描述、日常表現、特殊表現。導師評語分頁。選擇匯入學期，可決定與既有內容合併或覆蓋。
段考成績 Excel含座號、姓名、科目分數、班級名次、類組名次的段考表。成績表現分頁。選學期與一段考、二段考或期末考；系統會整理較強與較弱科目。
學測模擬考 Excel含級分、總級分、總分與排名的模考表。學測模考分頁。選第一次、第二次或第三次模考，再開始匯入。
社團資料 Excel社團名單、職稱、活動表現。社團活動分頁。選學期，確認姓名、座號、社團、職稱、表現欄位對應。
自評批次 Excel整班學生自評資料表。學生自評分頁。依姓名比對名冊，需先有學生名冊。選學期後開始匯入。
Google 問卷彙整 ExcelGoogle 表單回覆匯出的 Excel。學生自評、職位工作、補充欄位等。系統會猜測欄位對應，請務必確認姓名欄正確。
個別學生自評 .docx單一學生的 Word 自評表。該學生的學生自評分頁。系統會嘗試偵測姓名、座號與自評欄位；需選擇匯入到哪位學生與哪個學期。
班級備份 JSON本系統匯出的整班或單一學生 JSON。還原整班資料或匯入單一學生。整班備份會覆蓋目前資料；單一學生可覆蓋同名學生或新增。
降低出錯的做法：先匯入學生名冊，再匯入其他資料。因為大部分批次匯入都需要用「姓名」或「座號」比對學生，若名冊尚未建立，資料會找不到對應學生。

匯入前檢查清單

  • Excel 第一列或附近是否有清楚欄位名稱，例如姓名、座號、班級名次。
  • 學生姓名是否與名冊一致，避免同音異字、空白或暱稱。
  • 若同名學生存在，建議同時保留座號欄。
  • 匯入大量資料前先按「備份班級」。
  • 匯入後抽查 2 至 3 位學生，確認資料是否寫入正確分頁與學期。

六、評語草稿暫存區與 AI 潤飾

「評語草稿暫存區」位在學生資料表中，用來把本學期資料整理成正式評語前的草稿。它分成「文字描述評語草稿」與「特殊表現草稿」兩種。

按鈕功能操作說明
整合本學期資訊把目前學生該學期的自評、活動、成績、職位、社團與專題資料整合成草稿。先選對學期，再按此按鈕；如果資料太少，系統會提示沒有資料可整合。
AI 潤飾呼叫已設定的 Gemini、OpenAI 或 Claude API，把草稿改寫得更通順。需先按「AI 設定」填入 API 金鑰；若失敗，可改用複製功能貼到 AI 網站。
填入正式欄位把草稿寫入「導師評語」分頁的正式欄位。文字描述評語草稿會填入「文字描述評語」；特殊表現草稿會填入「特殊表現」。若正式欄已有內容，系統會確認是否覆蓋。
清除草稿刪除該草稿區文字。只清除草稿，不會刪除其他分頁資料。
提示詞選單選擇要 AI 採用的改寫方向。可選完整評語、精煉鼓勵、50 字以內、特殊表現描述、去除條列感等。
複製(含提示詞)複製提示詞與草稿全文。適合貼到 ChatGPT、Gemini 或 Claude 網站。
複製原文只複製草稿文字。適合貼到其他文件或自行編修。
ChatGPT / Gemini / Claude複製內容後開啟對應 AI 網站。開啟後在網站輸入框貼上即可。此方式不需要在本程式儲存 API Key。

AI 設定視窗

七、列印、匯出與備份

列印

  1. 先選取要列印的學生。
  2. 確認「檢視學期」是全部或指定學期。
  3. 按「列印」。
  4. 在跳出的選擇視窗勾選要包含的分頁。
  5. 按「列印」後，瀏覽器會開啟列印視窗。
  6. 可直接列印紙本，也可選「另存為 PDF」。

匯出 Excel

「匯出 Excel」會輸出目前選取學生一個 Excel 檔。檔名會包含座號、姓名與目前檢視學期。資料會依照一到九分頁的順序排列，自訂項目會接在後面。

匯出全班

「匯出全班」會建立一個 Excel 檔，每位學生一個工作表分頁。適合期末交接、導師留存或行政彙整。匯出前同樣可以勾選要包含哪些大項目。

匯出 JSON

右上方「匯出 JSON」會輸出目前學生的 JSON。左側「備份班級」會輸出整班 JSON。兩者用途不同：單一學生 JSON 適合個別轉移；整班 JSON 適合完整備份與還原。

八、日常操作建議

每週或每月

  • 補記學生幹部、服務、競賽與特殊表現。
  • 匯入社團或問卷資料後抽查幾位學生。
  • 按「備份班級」保存 JSON。

段考後

  • 匯入段考成績 Excel。
  • 檢查較強科目與較弱科目是否合理。
  • 若有重要學習轉折，補寫到導師評語或特殊表現。

期末前

  • 切換到該學期逐一檢查學生資料。
  • 用評語草稿暫存區整合文字描述評語。
  • 列印或匯出幾位學生測試格式。

期末完成後

  • 匯出全班 Excel 留存。
  • 備份整班 JSON。
  • 將備份檔放到雲端硬碟或安全資料夾。

九、常見問題

為什麼重新開啟網頁後資料還在？

因為資料存在同一台電腦、同一個瀏覽器的 localStorage。這讓系統不用登入也能保存資料，但也代表資料不是存在 GitHub 或雲端伺服器。

換電腦後看不到資料怎麼辦？

請在原電腦按「備份班級」匯出 JSON，再到新電腦開啟系統，按「匯入資料」選「班級備份 JSON」還原。

匯入 Excel 後很多學生找不到？

通常是名冊尚未建立、姓名不一致、座號欄未指定，或表頭列判斷錯誤。建議先匯入名冊，再確認 Excel 的姓名與座號欄位。

刪除欄位會只刪目前學生嗎？

不是。欄位結構屬於整班共用，刪除欄位會移除所有學生該欄內容。刪除前請先備份。

檢視學期選「全部」和選單一學期有什麼差別？

選「全部」時會顯示或匯出完整學期資料；選單一學期時，列印與匯出會以該學期為主。學測模考分頁以模考次數為主，不受學期篩選。

AI 潤飾失敗怎麼辦？

請確認 API Key 正確、模型名稱可用、帳號有額度，且瀏覽器沒有被 CORS 或網路限制擋住。若仍失敗，可按「複製(含提示詞)」再貼到 ChatGPT、Gemini 或 Claude 網站。

可以多人共同編輯同一份班級資料嗎？

此系統不是多人同步資料庫。每個瀏覽器各自保存資料。若多人要接力，請用 JSON 備份檔交換，但要注意版本覆蓋問題。

如何避免期末資料遺失？

每次大量匯入、欄位調整、評語完成或期末結束後，都下載整班 JSON。建議檔名保留日期，並另存到雲端硬碟。
